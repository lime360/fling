# Fling
A tiny self-hostable webring.

## Installation
```sh
$ git clone https://codeberg.org/lime360/fling
$ cd fling
$ pip3 install -r requirements.txt
$ python3 cli.py init # this is optional as fling automatically creates a database if it's not found
$ python3 app.py
```
